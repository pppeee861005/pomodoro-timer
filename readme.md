# 番茄鐘 Focus Web

這個專案是一個可愛風番茄鐘網頁，提供：

- 可自訂工作 / 短休息 / 長休息時間
- 卡通番茄插畫與焦點提示
- 可調整的進度條、控制按鈕與統計數字
- 設定面板與暫時通知提示

### 開始使用

1. 進入專案資料夾。
2. 啟動靜態伺服器，例如：
   ```bash
   python -m http.server 8000
   ```
3. 使用瀏覽器開啟 `http://127.0.0.1:8000` 進行測試。

### 技術堆疊

- 原生 HTML / CSS / JavaScript
- 利用 LocalStorage 保存設定與番茄數
- Web Audio API 播放提示音

### 注意事項

- 設定面板點擊背景或右上角 ✕ 就會關閉。
- 計時結束會顯示 toast 提示，不再使用 alert 彈窗。
